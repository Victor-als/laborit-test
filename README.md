# FinTechX Chat — Assistente Virtual Inteligente

Interface de chat inteligente desenvolvida para melhorar o atendimento ao cliente da FinTechX, com foco em experiência do usuário, simplicidade e respostas automatizadas com IA.

---
## 🚀 Funcionalidades

* Interface de chat moderna e responsiva
* Envio e recebimento de mensagens em tempo real
* Integração com modelo de linguagem (IA)
* Indicador visual de resposta (loading animado)
* Scroll automático das mensagens
* Cards inicais com capacidades do assistente
* Layout inspirado em produtos reais de chat
* Tratamento básico de erros

---

## 🧠 Objetivo do projeto

Resolver problemas reais da FinTechX:

* Falta de personalização no atendimento
* Respostas lentas ao cliente
* Experiência pouco intuitiva

A solução propõe um **assistente virtual inteligente** capaz de responder dúvidas comuns de forma rápida, acessível e eficiente.

---

## 🛠️ Tecnologias utilizadas

| Tecnologia                    | Uso                     |
| ----------------------------- | ----------------------- |
| React                         | Interface do usuário    |
| TypeScript                    | Tipagem e segurança     |
| Tailwind CSS                  | Estilização             |
| Vite                          | Build e desenvolvimento |
| API de IA (Groq / OpenRouter) | Respostas inteligentes  |
| Vercel                        | Deploy                  |
| GitHub Actions                | CI/CD                   |

---

## ⚙️ Instalação e execução

```bash
# Instalar dependências
npm install

# Rodar o projeto
npm run dev
```

Acesse: http://localhost:5173

---

## 🔐 Variáveis de ambiente

Crie um arquivo `.env` baseado no exemplo:

```
VITE_API_KEY=your_api_key_here
```

---

## 🧪 Build

```bash
npm run build
```
